Exercícios aula 03

  1. Imprima na tela os número de 1 a 10
  2. Calcule a soma dos números de 1 a 50
  3. Para um número inteiro, faça um loop while que incremente-o de 1 000 em 1 000, imprimindo seu valor, até que seu valor seja menor ou igual 100 000. (cem mil)
  4. Faça um script utilizando o comando while que mostra uma contagem regressiva na tela. O valor de início deve ser definido pelo usuário. Ao final, deverá ser mostrada uma mensagem “FIM!”
  5. Escreva um loop do while que exiba os números de 0 ao número passado pelo usuário (utilize prompt).
  6. Escreva um algoritmo que exiba todos os números pares de 0 ao número passado pelo usuário (utilize prompt).
  7. Escreva um script que some os números ímpares contidos em um intervalo definido pelo usuário. O usuário define o valor inicial do intervalo e o valor final deste intervalo e o programa deve somar todos os números ímpares contidos neste intervalo. Caso o usuário digite um intervalo invalido (começando por um valor maior que o valor final) deve ser escrito uma mensagem de erro na tela, “Intervalo de valores invalido” e o programa termina.>
